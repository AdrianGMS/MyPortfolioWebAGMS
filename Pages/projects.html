The results of the recognition of one computer lab

I developed this project as my thesis during my last year of university. It's an application built using React Native for the app interface, Python for facial recognition using the face-recognition python library, and Firebase as the database and backend for the app. This is where attendance records, photos with results, and information for model training are stored.

The app is simple. Professors can make human errors when taking attendance or even when they are supervising a midterm or final exam in an unfamiliar classroom. This is why they only need to access the application, where each student already has their faces registered and assigned to different courses. The professor simply has to log in, take a photo, upload it, and attendance will be automatically recorded.

Link to github project: https://github.com/AdrianGMS/FaceCounter
